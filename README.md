# AI-Powered Study Buddy 📚✨

An AI-powered learning assistant that helps students understand complex concepts, summarize notes, generate quizzes, create flashcards, and build personalized study plans using modern Large Language Models.

## 🎯 Features

### 1. **Topic Explainer**
- AI-powered explanations for any topic
- Simple beginner-friendly explanations
- Real-world examples and use cases
- Helps students understand difficult concepts quickly

### 2. **Notes Summarizer**
- Upload PDF and text files
- Extract study material automatically
- Generate concise summaries
- Highlight important points and concepts

### 3. **Quiz Generator**
- Generate MCQ questions
- Generate True/False questions
- Generate short-answer questions
- Provides answers and explanations
- Test understanding of any topic

### 4. **Flashcard Generator**
- Create flashcards from notes
- Convert study material into revision cards
- Improve memorization and quick revision

### 5. **Study Planner**
- Create personalized study schedules
- Enter exam date and available hours
- Generate optimized learning plans
- Manage preparation effectively

### 6. **Ask Questions from Notes**
- Upload learning materials
- Ask questions based on uploaded content
- Get context-aware answers
- Improve concept clarity
