## 🚀 QUICK START COMMANDS

### Setup
```bash
# Automatic setup
python setup.py

# Manual setup
python -m venv venv
source venv/bin/activate
pip install -r requirements.txt
```

### Configuration
```bash
# Copy environment template
cp .env.example .env

# Edit and add your API key
# GEMINI_API_KEY=your_key_here
```

### Run Application
```bash
# Windows
run.bat

# macOS/Linux
bash run.sh

# Direct
streamlit run app.py
```

### Access Application
```
http://localhost:8501
```

### Run Tests
```bash
pytest tests/ -v
pytest tests/ --cov=src/
```
